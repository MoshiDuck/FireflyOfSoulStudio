## 🔗 Dépôt GitHub

https://github.com/MoshiDuck/FireflyOfSoulStudio

### Réinitialiser

1. Supprimer l'historique Git local + recrée un dépôt Git vierge
    ```bash
   Remove-Item -Recurse -Force .git
   ```

2. Init git
    ```bash       
   git init
   ```

3. Mettre main
   ```bash  
   git checkout -b main
   ```

4. Lier au dépôt distant GitHub
    ```bash
   git remote add origin https://github.com/MoshiDuck/FireflyOfSoulStudio
   ```
---

### Ajouter / Modifier

1. Ajoutez les dossiers
    ```bash
    git add .
    ```
2. Commit
    ```bash
    git commit -m "[FAIT] - Site debut"
    ```

3. Version
   ```bash
   git tag 0.0.0.1
   ```

4. Force le push
    ```bash
    git push --force origin main
    ```
